HTML Basics

The "frontend" of a webpage consists of:

  • HTML = HyperText Markup Language
    • content
  • CSS = Cascading Style Sheet
    • style
  • JS = Javascript
    • interactivity

Vocabulary & Syntax

  • Element have the following syntax: < tag attribute="value" > content < /tag >

Block vs. Incline

  • Block-level elements always start on a new line and take up the full width avaliable
  • Incline-elements do not start on a new line and only takes up as much width as necessary

Block-Level Elements

  • Heading: < h1 > < /h1 > through < h6 > < /h6 >
  • Horizontal Rule (Line): < hr >
  • Paragraph: < p > < /p >
  • Ordered List: < ol > < /ol >
  • Unordered List: < ul > < /ul >
  • List Items: < li > < /li >
  • Form: < form > < /form >
  • Div (Container): < div > < /div >

Incline Elements

  • Line Break: < br >
  • Image: < img src="URL" >
  • Link (Anchor): < a href="URL > < /a >
  • Code: < code > /code >
  • Bold: < strong > < /strong >
  • Italicized: < em > < /em >
  • Span (Container): < span > < /span >
